# x-forth + 中文语法关键词

基于英文的程序语言的特点是，大量依赖标点符号。
基于中文的程序语言，是否应该避免大量使用标点符号？

    函数（参数，参数）；

如果应该避免，那么类 forth 的语法就是语法设计的合理方案之一。

选［］来定义局部变量，对应英文的 []。

语法关键词选择：

1. 函数 常量 变量
2. 定义 常量 变量
3. 定义 定义常量 定义变量

选 2。

注释选择：

1. -- 注释
2. ＃ 注释

选 1。

例子：

```forth
定义 平方［数］
  数 数 相乘
结束

定义 阶乘［数］
  数 零？ 则
    一
  否则
    数 减一 阶乘 数 相乘
  然后
结束
```

```forth
・定义 平方［数］
  数 数 相乘
・结束

・定义 阶乘［数］
  数 零？ ・则
    一
  ・否则
    数 减一 阶乘 数 相乘
  ・然后
・结束
```

```forth
＠定义 平方［数］
  数 数 相乘
＠结束

＠定义 阶乘［数］
  数 零？ ＠则
    一
  ＠否则
    数 减一 阶乘 数 相乘
  ＠然后
＠结束
```
