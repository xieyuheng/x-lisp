选〔〕或［］来定义局部变量。
呼应英文中的 `[]`。

语法关键词选择：

1. 函数 常量 变量
2. 定义 常量 变量
3. 定义 定义常量 定义变量

选 2。

注释选择：

1. -- 注释
2. ＃ 注释

选 1。

例子：

```forth
定义 平方［数］
  数 数 相乘
结束

定义 阶乘［数］
  数 零？ 则
    一
  否则
    数 减一 阶乘 数 相乘
  然后
结束
```
