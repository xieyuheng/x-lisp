《》
﹝﹞
（）
〔〕
【】
「」

语法关键词选择：

- 函数 常量 变量
- 定义 定义常量 定义变量

注释选择：

-- 注释
＃ 注释

```forth
  否则
    数 减一 阶乘 数 相乘
  然后
结束
```
